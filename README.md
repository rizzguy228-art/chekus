# Чекушка Mod — Minecraft Forge 1.20.1

## Что делает

Добавляет предмет **Чекушка Заветная №1** в раздел **Еда и напитки** творческого инвентаря.

### Эффекты при употреблении (1 минута):
| Эффект | Уровень |
|--------|---------|
| ⚔️ Сила | III |
| 🏃 Скорость | IV |
| 🌑 Слепота | I |

- Восстанавливает **4 полоски голода** (8 единиц)
- Можно есть всегда (даже с полным голодом)
- Стакается до 16 штук

### Рецепт крафта:
```
[ ]  [Стеклянная бутылка]  [ ]
[ ]  [Пшеница]             [ ]
[ ]  [Стеклянная бутылка]  [ ]
```

## Сборка

1. Скачай [Forge MDK 1.20.1](https://files.minecraftforge.net/net/minecraftforge/forge/index_1.20.1.html) → **Mdk**
2. Распакуй MDK, скопируй файлы этого мода **поверх** (с заменой)
3. Собери:
   ```bash
   # Windows
   gradlew.bat build

   # Linux / Mac
   ./gradlew build
   ```
4. Готовый `.jar` — в `build/libs/chekushka-1.0.0.jar`
5. Положи в папку `mods` вашего `.minecraft`

**Требования:** JDK 17, Forge 47.3.0+
